Ambition project

Dit is een game die ik helemaal zelf heb gemaakt
met C# in Unity in de tweede periode van vorig schooljaar,
de opdracht was om iets te maken dat overeenkomt met je ambitie.
Voor mij is dat het bouwen van games,
dus begon ik na te denken over wat voor soort game ik wilde maken,
ik had niet veel tijd, dus hield het eenvoudig en maakte een 2D endless runner.
De game is volledig compleet, inclusief een homescreen, backgroundmusic, enz.
Ik heb ook de visuals van de main character zelf gemaakt.
Ik heb geleerd dat het belangrijkste bij het bouwen van een game
is om altijd een goede planning te hebben
voor alles wat je wilt doen wanneer je beperkte tijd hebt.
want al snel genoeg kwam ik erachter
dat ik veel dingen moest doen in de korte tijd die ik had
omdat ik geen goede planning had.
